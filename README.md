# Assisted Workflow

Este projeto documenta uma metodologia de trabalho orientada por IA, baseada no fluxo definido em `workflow/README.md`.

## Objetivo

Registrar e comparar a aplicacao da metodologia com diferentes ferramentas de IA, mantendo um processo consistente e rastreavel.

## Fluxo oficial

A ordem das etapas esta em `workflow/README.md`:

1. `/requirements`
2. `/prepare`
3. `/planning`
4. `/implement`
5. `/commit`
6. `/summarize`

## Estrutura do repositorio

- `workflow/requirements/`: requisitos por tarefa
- `workflow/plans/`: planos de implementacao
- `workflow/summaries/`: resumos finais e evidencias

O `.gitignore` foi configurado para ignorar o conteudo gerado nessas subpastas, preservando a estrutura com arquivos `.gitkeep`.
